# AI-Quiz

The AI Quiz Tool is an intelligent web-based application that leverages AI to generate and evaluate quizzes dynamically. Users can attempt quizzes and receive instant feedback, including scores, correct answers, and concise explanations with reference links for further learning. The tool enhances user experience with a clean UI, structured result tables, and interactive features, making it ideal for self-assessment and knowledge reinforcement.

- This tool requires an API key to access the AI engine.
- To generate your API key, visit https://groq.com/ and create one from your account.
- Provide the API key either within your request or by passing it as a URL query parameter (e.g., ?key=YOUR_API_KEY).

Test your knowledge with an AI-powered quiz engine that not only evaluates your answers but also explains them—helping you learn smarter, not harder.
